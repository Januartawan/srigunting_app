9. 🎨 DESIGN IMMUTABILITY
   - Jangan merubah atau memodifikasi desain/tampilan UI yang sudah ada, kecuali secara eksplisit diminta.
   - Setiap perubahan hanya boleh pada data, logika, atau alur, tanpa mengubah struktur, layout, warna, atau komponen visual yang sudah ada.
   - Jika ada kebutuhan untuk mengubah desain, harus ada instruksi atau persetujuan eksplisit dari pihak terkait.
